## 2025-10-27 - Accessible Skip Links in Neon/Glass UI
**Learning:** Standard "skip to content" links can break immersion in highly stylized portfolios.
**Action:** Style the skip link as a UI element (glass/neon) that slides in, treating it as a hidden "power user" feature rather than an ugliness to be hidden.
